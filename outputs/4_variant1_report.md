# Робота 4 — Варіант 1: Частки й довірчі інтервали

Скрипт `4_звіт.R` обчислює одночасні Wilson Confidence Intervals (з корекцією Bonferroni-подібним способом) для частки чоловіків у вибраних штатах та виконує порівняння з гіпотезою p = 0.51.

**Результати:**

- Результати збережено у `outputs/4_variant1_results.csv`.
- Графік одночасних інтервалів довіри збережено у `plots/report4/4_variant1_CIs.png`.

![Simultaneous Wilson CIs (Variant 1)](../plots/report4/4_variant1_CIs.png)

> Примітка: запустіть `4_звіт.R` після додавання файлу `data/ChGendUS2023.csv` для генерації таблиць та графіків.
